Black And White

A Classic Android Piano Game

  • created: 07/07/2014
  • latest update: 07/07/2014
  • by: Melvin Zammit
  • www.neurondigital.com/
  • email: melvin@neurondigital.com

This is a classical Black and White game template for Android. The aim of the user is to tap on the Black tiles as fast as possible, while keeping his fingers away from the white tiles. The game includes 4 modes: Classic, Arcade, Zen and Bomb. Thanks for buying this Template!

Step By Step Customisation

Step 1 - Make sure you have Eclipse with the latest Android SDK installed on your system.

Step 2 - Import the 'CODE - Import me in eclipse' file into Eclipse. (3 files should be imported: google-play-services_lib, BaseGameUtils, Black and White)

If you see an error at this point, its because one of the 2 libraries is not loaded well. Right click on Black and White. Goto Properties|Android. In the Library section make sure to have the google play services library and BaseGameUtils loaded (green). Click add to add them.

Step 3 - Replace any PlaceHolder Images

In Eclipse you can change any images you wish. At least make sure to change the icon. To replace the images just drag and drop your new images in res/drawable.

Step 4 - Make sure to rename the App Package id

1.    First rename the package id in the AndroidManifest.xml:

<manifest xmlns:android="http://schemas.android.com/apk/res/android"
package="com.yourcompany.AppName"
android:versionCode="1"
android:versionName="1" >

2.    Rename the package id from the 'Package Explorer' window in eclipse. Just expand the Black and white File, expand the underlying 'src' file and click on tha file named 'com.neurondigital.blackandwhite'. Then just go to file| Rename.

If you don't follow these steps or do step 2 first, eclipse will generate R. references in your code, which you would have to delete manually, else there would be errors generated.

Step 5 - Create an interstitial Ad on AdMob and paste the AD unit Id in the Strings.xml file.

The Ad displays once before gameover. Make sure to add this to start making money from your App.

Step 6 - Create an Banner Ad on AdMob and paste the AD unit Id in the Strings.xml file (otional).

The Ad displays in the bottom of the screen. Ideally it's either the banner or the interstitial ad, so you may wish to leave one of the IDs in strings.xml empty.

Step 7 - Google Play Services Leaderboard (optional).

The game template can connect to a google play leaderboard when opened and upload the new score after each gameover. To enable this just create a leaderboard on Google Play Store and put the generated App id and leaderboard ids in strings.xml. You need to create 4 leaderboards, one for each mode's top score.

  <string name="app_id">PUT APP ID HERE</string>
  <!-- Google Play services leaderboard Ids - You need to create 4 leaderboards in google Play developer console, one for each top score. Then just get each one's id and put it here -->
  <string name="Classic_leaderboard_id">PUT ClASSICAL LEADERBOARD ID HERE</string>
  <string name="Arcade_leaderboard_id">PUT ARCADE LEADERBOARD ID HERE</string>
  <string name="Zen_leaderboard_id">PUT ZEN LEADERBOARD ID HERE</string>
  <string name="Bomb_leaderboard_id">PUT BOMB LEADERBOARD ID HERE</string>

If this is not done, the top score icon fill not be shown. Read more about leaderboards on the official docs: https://developers.google.com/games/services/common/concepts/leaderboards

Step 8 - Export your App as a .APK for Google Play

Personalising App

TODO Tags

In the code you will find TODO tags. Make sure to check them out. They indicate the places that you usually need to change something, although in this app most of the things that would be changed are all in the strings.xml file.

 

Change Colors

Open Strings.xml. In the code search for the following snippet. Change the color codes, to change the respective colors.

    <!-- Primary Colors -->
    <color name="white">#ffffffff</color>
    <color name="black">#ff000000</color>
    <color name="red">#ffDC574E</color>
    <color name="green">#ffA7BD5B</color>
    <color name="grey">#ffB8B8B8</color>
    <color name="trans_black">#aa2F3738</color>

Black tile Patterns

The black and white tiles are generated from a predefined array. The array currently contains 2 patterns, and one of them is selected randomly on game start. To change the pattern just open maingame.java line114:

int[][][] classic_arcade_zen_pattern = new int[][][]
{
   {
      { 0, 1, 1, 1 },
      { 1, 0, 1, 1 },
      { 1, 1, 0, 1 },...

In the code excrept above you can see the first 3 tile rows generated. The 0 indicates a black tile while the 1 indicates a white tile.

Font sizes

Feel free to change any font sizes from MainGame.java, line 292.
Example to increase the title font size just increase the 35 in the code below
 
Title_Paint.setTextSize(dpToPx(35));
Title_Paint.setAntiAlias(true);
Title_Paint.setColor(getResources().getColor(R.color.black));
Title_Paint.setTypeface(Orbitron);
 

Editing Images

Most of the images have with them included the .ai file. So feel free to edit them as you like. Remember to keep the same image resolution before replaceing it with the one in eclipse.
